The Thrust library

Day: Monday, November 11
Time: 2:10 PM - 3:00 PM
Place: Snedecor Hall 2113

The Thrust Library is a freely-available GPU-accelerated implementation of the C++ Standard Template Library (STL). Thrust provides a flexible, high-level interface and powerful GPU-accelerated algorithms that include sorts, scans, transformations, and reductions. As with CUBLAS and CULA, Thrust micromanages all the GPU parallelism so the user does not have to.
